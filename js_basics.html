4 Ways to Declare a JS Variables

Declaring variable with "var" keyword:

        var x = 1;
        var y = 2;
        var z = x + y;

Declaring variable with "let" keyword

        let a = 1;
        let b = 2;
        let c = a + b;

Undeclared variables

        g = 1;
        k = 2;
        m = g + k;

Mix use of let and const keyword

       
        const price1 = 5;
        const price2 = 6;
        let total = price1 + price2;
        document.getElementById("var4").innerHTML = 'The total is: ' + total;
